Getting Started:
---------------

	# Make sure you have mongodb installed 
		-if not install using homebrew

	# Make sure mongod is running	
	
	$ cd /path/to/your/folder

	$ npm install
	
	$ mkdir tmp
	
	$ mkdir /public/uploads	

	$ mkdir /public/uploads/clients	

	$ node app.js
